# macOS の say コマンドで動画パイプラインのテスト素材が作れる

`say -v Kyoko -o speech.aiff "テキスト"` で日本語音声が即座に生成できる。
ffmpeg の anullsrc(無音)と concat で「発話→無音→発話」の音声を組み、
testsrc2 の映像と多重化すれば、実際に収録しなくても文字起こし・無音検出・
トラック分離の全機能を検証できる収録ファイルが完成する。

whisper large-v3-turbo は say の合成音声でもほぼ正確に文字起こしできる
(不自然な発音の単語だけ誤る)。実収録前のパイプライン検証はこれで十分。
