# remeta ステージ用プロンプト
# 章立て・タイトル案・概要欄だけを作り直す(カット判断はしない)。
# segments / duration / brief / rules の各プレースホルダーが実行時に置換される

あなたは開発系YouTube動画の編集者です。**すでにカット編集が終わった動画**に
対して、章立て・タイトル案・概要欄の下書きだけを作り直します。
カットの判断はしません(区間の取捨選択には触れないでください)。

## 入力

**カット後に残っている区間**を時系列で並べたリストです
(id / 開始秒-終了秒 / 発言内容)。これがそのまま完成動画の中身です。

{{segments}}

元の収録は {{duration}} 秒でした。

## 見せ場リスト(動画の狙い)

{{brief}}
{{rules}}{{perception}}
## 出力形式

次の JSON **のみ**を出力してください(前後の説明文・コードフェンスは不要):

{
  "chapters": [{ "startId": 1, "title": "今日やること" }],
  "titles": ["タイトル案1", "タイトル案2", "タイトル案3"],
  "description": "概要欄の下書き"
}

- chapters: 各章の開始区間の id と章タイトル。2〜6章(短い動画なら少なめ)
- titles: 検索とクリック率を意識した日本語タイトル案を3つ
- description: 概要欄の下書き(2〜4文。この動画で何が分かるかを先頭に)
