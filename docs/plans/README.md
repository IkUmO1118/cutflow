# plans/ — 実装設計書(単発・完了記録)

ここに置くのは、単発の機能を実装する前に書いた設計書。**ほぼ全て実装済み・
マージ済み**で、削除やアーカイブ化はせずコードの変更履歴としてそのまま残して
いる。書いた時点のスナップショットなので実装後に細部が変わっていることも
ある(正はコード)。

生きているロードマップ/継続中のプログラムは `docs/programs/` を見る。
そちらは §番号で追記され続ける文書で、単発の設計書とは別カテゴリ。個別の
設計書の冒頭にある「親ドキュメント: `docs/programs/....md`(母艦)」が
その参照。

`regression/` は母艦の回帰測定に使う台帳(進行中の計測ツール。単発の設計書
ではない)。
