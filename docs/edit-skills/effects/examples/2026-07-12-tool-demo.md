## この記録の読み方

この収録は **`effectReasonId` / overlaysの`reasonId`導入前**に編集された。
したがって実在する`overlays.json`の4件には分類idが保存されていない。以下の
`effectReasonId`は、元秒・rect・隣接関係という実測値を変えず、EP4でrecipeへ
**後付け分類した分析**である。生成当時にLLMがこのidを返した記録ではない。

`start` / `end` は元収録の秒、`rect` は出力px。`anchorId`はこの文書内の照合用番号で、
当時の`plan-effects`候補番号ではない。座標・時刻をrecipeから推測した例ではなく、
承認済みoverlaysの値をそのまま記録する。

## この記録から言える範囲

実測で直接言えるのは、zoom 4件の元秒・rectと、#2/#3が445.32秒で接する連鎖で
